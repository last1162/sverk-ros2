## 🚀 What This App Does

sverk-ros2 is a Windows app for working with PX4-powered drones through ROS 2. It helps you connect your drone control setup, run basic commands, and manage flight tasks from one place.

Use it if you want a simple way to:

- Connect to a PX4-based drone setup
- Send control commands through ROS 2
- Start and stop common drone actions
- Test your drone link from a Windows PC

## 💻 Before You Start

Use a Windows PC with:

- Windows 10 or Windows 11
- A stable internet connection
- At least 4 GB of RAM
- 500 MB of free disk space
- Admin access for setup if Windows asks for it

You may also need:

- A PX4-powered drone or test setup
- A ROS 2 compatible network link
- A USB cable, serial link, or local network connection, based on your setup

## 🛡️ Troubleshooting

### The app does not open

- Try running it as an admin
- Check whether Windows blocked the file
- Re-download the release file if it looks damaged

### Windows shows a security prompt

- Select More info if needed
- Confirm you want to run the file
- Make sure you downloaded it from the release page

### The drone does not connect

- Check power on the drone or simulator
- Confirm the cable or network link
- Restart the app
- Restart the PX4 side if needed
- Make sure your ROS 2 setup is active

### The app opens, but nothing happens

- Check that the drone link is live
- Confirm the correct device or node is selected
- Close the app and open it again
- Try a fresh download from the release page
